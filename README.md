# Codex Skills Reference

Reusable Codex skills for Apple development (design system, SwiftUI, concurrency, security, etc.).

This repo's primary purpose is **skills + command references**. The original Swift package was exploratory and is not actively used.

## Skills
See `.agents/skills/` (and manifest.json).

Install for Codex with:
```bash
./Scripts/install-skills.sh
```

## MCP & Commands
See `MCP.md` for XcodeBuildMCP, useful prompts, and bootstrapping.

Point Codex here when bootstrapping apps with these skills.
