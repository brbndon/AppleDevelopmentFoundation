# Design System

Use `FoundationTokens` semantic spacing, radius, and role values rather than scattering fixed visual values. Components choose neutral system colors and system text styles, so applications may apply their own brand tint at the app layer. Components use native controls, accessible labels, scalable layout, semantic status text, and standard materials. Test in light/dark mode, increased contrast, large Dynamic Type, Reduce Motion, VoiceOver, keyboard-only macOS navigation, hover/pointer behavior, and iOS touch sizing before broadening the API.
