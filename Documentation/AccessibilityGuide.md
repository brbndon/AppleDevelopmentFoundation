# Accessibility Guide

All reusable components must use system text styles or scalable metrics, semantic colors plus non-color status cues, native Buttons for actions, and clear labels/hints for icon-only controls. Review VoiceOver grouping/order, text at accessibility sizes, full keyboard access and focus on macOS, iOS touch targets, dark appearance, Increased Contrast, Differentiate Without Color, and Reduce Motion. Include accessibility behavior in examples and do not hide meaningful images from assistive technologies.
