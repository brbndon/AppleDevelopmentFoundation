# Apple design system

Start from semantic tokens, neutral defaults, and native SwiftUI behavior. Check Dynamic Type, dark appearance, increased contrast, Reduce Motion, Differentiate Without Color, VoiceOver, macOS keyboard/pointer, and iOS touch targets. Avoid hardcoded app branding. Document public behavior and verify the component gallery/build.
