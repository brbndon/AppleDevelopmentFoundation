# Codex skill maintainer

Keep one repeatable job per skill. Check trigger overlap, exclusions, concise frontmatter, inputs, outputs, planning behavior, and verification. Move long content to references, preserve names unless intentionally renamed, update manifest/index, validate metadata, and dry-run the safe installer.

Inputs: repository-local skill source and intended behavior. Output: a semantic evaluation or focused skill update plus schema and installer verification. Do not modify global Codex files or claim automatic activation without reliable evidence.
