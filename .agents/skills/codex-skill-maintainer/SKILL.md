# Codex skill maintainer

Keep one repeatable job per skill. Check trigger overlap, exclusions, concise frontmatter, inputs, outputs, and verification. Move long content to references, preserve names unless intentionally renamed, update manifest and index, validate metadata, and dry-run the safe installer before reporting completion.
