# Apple accessibility review

Inspect VoiceOver labels/grouping, Dynamic Type, focus and keyboard order, touch targets, contrast, Reduce Motion, Increased Contrast, Differentiate Without Color, icon-only labels, and platform differences. Return actionable findings, add focused tests/examples where possible, and verify on both supported platforms when public behavior differs.
