# FoundationTasks

An iOS 17+ SwiftUI host app created with Xcode. It demonstrates local-package integration by using `DesignSystem` for semantic UI, `FormKit` for task-title validation, `NavigationKit` for app-owned routing, and `PersistenceKit` to create its SwiftData container.

Open `FoundationTasks.xcodeproj` in Xcode and run the `FoundationTasks` scheme. The local package reference resolves to the repository root.
