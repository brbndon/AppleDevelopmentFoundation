# Apple Development Foundation

This repository’s **product** is a set of reusable **Codex skills** and command references for Apple development (iOS 17+, macOS 14+). The Swift package under `Sources/` and the old Vite site under `Website/` are **archived** exploratory work — do not expand them unless the user explicitly asks.

Use this Blume site as the **agent-facing playbook**: how to install skills, route work, call MCPs, bootstrap consumer apps, and verify changes.

## Start here

:::tip For coding agents
1. Read [Quickstart](/quickstart).
2. Follow the [session workflow](/workflow).
3. Route with `$apple-development-foundation` when the task is broad or a new chat needs a skill shortlist.
4. Prefer **XcodeBuildMCP** over raw `xcodebuild` / `simctl` — see [XcodeBuildMCP](/tools/xcodebuildmcp).
:::

## Core rules (do not skip)

- Skills apply to the **consumer workspace** (the app or package the user is building), not this repo’s archived modules.
- Keep skills neutral: no business models, branding, secrets, user-specific paths, or hidden network behavior.
- Use Swift 6, SwiftUI, structured concurrency, and focused views. Do not add a view model without a state-ownership or testability reason.
- All SwiftUI UI must support Dynamic Type, VoiceOver, keyboard, contrast, Reduce Motion, Differentiate Without Color, and labels for icon-only controls.
- Never log credentials, tokens, private content, raw imports, complete sensitive paths, or security-scoped URLs.
