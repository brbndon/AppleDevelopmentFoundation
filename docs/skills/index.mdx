## Inventory

Deep procedure lives in each skill’s `SKILL.md` — prefer reading that over forking long instructions into this site. Paths are relative to the foundation repository root.

| Skill | Path | Use for | Do not use for |
| --- | --- | --- |
| `apple-development-foundation` | `.agents/skills/apple-development-foundation/SKILL.md` | Reference-first child-skill routing; explicit foundation audits | Replacing child instructions; implied audits/verification |
| `codex-bootstrap` | `.agents/skills/codex-bootstrap/SKILL.md` | New iOS/macOS SwiftUI projects in the **consumer** workspace | One-off screens; archived package work in this repo |
| `apple-platform-planner` | `.agents/skills/apple-platform-planner/SKILL.md` | Planning a platform feature | Implementation-only requests |
| `swiftui-tab-navigation` | `.agents/skills/swiftui-tab-navigation/SKILL.md` | App-level tabs / system Liquid Glass tab chrome | Page-style tabs, segmented controls, business screens |
| `swiftui-component-author` | `.agents/skills/swiftui-component-author/SKILL.md` | Reusable SwiftUI components | App-specific screen copy |
| `apple-design-system` | `.agents/skills/apple-design-system/SKILL.md` | Design-system tokens and components | Application branding |
| `ios-macos-platform-adaptation` | `.agents/skills/ios-macos-platform-adaptation/SKILL.md` | Divergent iOS vs macOS behavior | Shared identical behavior |
| `swift-package-module-author` | `.agents/skills/swift-package-module-author/SKILL.md` | Independently importable SPM module in the active workspace | App-local feature; archived package here |
| `reusable-code-extractor` | `.agents/skills/reusable-code-extractor/SKILL.md` | Proven app code → consumer shared module | Speculative abstraction; landing code in this repo |
| `swift-concurrency-review` | `.agents/skills/swift-concurrency-review/SKILL.md` | async/await, actors, cancellation | Unrelated styling |
| `apple-accessibility-review` | `.agents/skills/apple-accessibility-review/SKILL.md` | Reusable UI accessibility | Non-UI services |
| `apple-security-privacy-review` | `.agents/skills/apple-security-privacy-review/SKILL.md` | Files, logging, permissions, sensitive data | Visual-only polish |
| `swift-testing-verification` | `.agents/skills/swift-testing-verification/SKILL.md` | Verification after meaningful code changes | Planning-only work |
| `maestro-apple-app-testing` | `.agents/skills/maestro-apple-app-testing/SKILL.md` | End-to-end Maestro UI testing | Non-testing product implementation |
| `codex-skill-maintainer` | `.agents/skills/codex-skill-maintainer/SKILL.md` | Creating/updating skills in this repo | Ordinary product code changes |
