# Archived exploratory work

| Path | Preserved content |
| --- | --- |
| `Package.swift`, `Sources/`, `Tests/` | Exploratory Swift package and tests |
| `Apps/`, `Examples/` | Package demonstrations and app experiments |
| `Website/` | Superseded Vite marketing site |
| `Documentation/` | Historical package architecture and policy notes |
| `Scripts/` | Package, demo-site, and environment tooling |

Archived implementation files were moved without refactoring or modernization. Run archived scripts from this directory only when work on the archived material is explicitly requested. Live product guidance and verification remain at the repository root; see [`../ARCHIVE.md`](../ARCHIVE.md).
